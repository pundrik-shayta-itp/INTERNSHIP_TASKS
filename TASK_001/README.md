# My Website

This project is a simple website built using semantic HTML, CSS, and JavaScript. It includes a homepage, an About page, and a Contact page.

## Project Structure

```
TASK_001
├── src
│   ├── index.html        # Main HTML file for the homepage
│   ├── pages
│   │   ├── about.html    # About page with information about the website or organization
│   │   └── contact.html   # Contact page with a form or contact details
│   ├── css
│       └── styles.css     # CSS styles for the website
└── README.md              # Documentation for the project
```

## Technologies Used

- HTML5
- CSS3
